PSV3.65破解+固化

Specia1Z · 2020-7-28 · 次阅读

准备

1.首先在索尼官网申请一个全新的psn账号，不要使用你原来psv或者ps4使用的存有数字游戏的正版账号，使用正版账号破解容易被封，申请一个专用于破解的空psn账号，即使被封也没关系。

2.如果是从未破解的新机器可以跳过2，如果是原先安装过双固化的人重新破解或者没事干折腾，记得在格式化前先恢复一键固化VITASHELL2.02和工具箱1.64的公园和日历，然后再卸载enso_365固化，必须按照顺序卸载，如果先卸载了enso_365破解就失效了，就恢复不了公园和日历了，格式化后会无法回到官方状态。

主机端

1.做好备份，初始化主机，格式化存储卡。

2.进入系统后连接WiFi，登录破解专用PSN账号。长按PS键打开快捷菜单，确认飞行模式和蓝牙已关闭，打开把设定里的USB充电关闭。

电脑端

1.电脑安装QCMA软件(er50)。

2.安装驱动QcmaDriver_winusb。

3.打开Qcma，任务栏会出现图标。

4.下载FinalHE_v1.92_vtiashell_win32(pbn0),解压后打开FinalHE,右上角下拉菜单选择简体中文(不要勾选将h-encore瘦身到约7mb),等待备用。

破解

1.PSV用USB线连接电脑，PSV上打开内容管理——复制内容——个人电脑——USB连接。然后会出现如下图所示画面，PSV就可以放着不动了。

2.FinalHE应该如下图所示，如果没有连接上就重试一下，多次之后还是不行，可以去下载PSV内容管理助手，在进行连接。

3.点击开始，等待完成。下图就表示已完成

4.在电脑的计算机里面找到用户文档的文件夹打开，找到PSV的文件夹，然后找到APP文件夹打开。会发现代表机子代码的空文件夹，里面的文件夹可以存放应用程序或者视频，复制进去之后，Qcma图标右键刷新数据库，然后就可以将文件复制到PSV上，即电脑右下角QCMA点击刷新数据库后，在PSV内容管理页面选择个人电脑——PSV——应用程序——PS Vita选项，复制程序，返回桌面就会出现该程序的气泡。

5.回到PSV，在没有中断usb连接状态的情况下（如中断，重新连接usb，主页打开内容管理——复制内容）直接在复制内容项目下点击电脑——psvita——应用程序——psvita，勾选h-encore2和vitashell点击右下角复制——ok，等待复制完成后撕页退出。

6.主页第三页出现h-encore2和vitashell气泡，打开h-encore2气泡，进入后选择第二项 install henkaku ，按o键，执行完毕后选第一项 exit 按o键回到主页。

7.进入PSV设定界面，打开HENkaku设置，勾选启用不安全自制软件，然后重启PSV。

8.进入vitashell，连接usb数据线，按select键连接usb，将enso_ex_v4.1_24x24_3.65_vs0.vpk或enso_ex_24x24-ur0.vpk(vs0和ur0的区别在于前者可以格式化破解不失效后者格机后需要重新破解，按照自身喜好选择)和一键固化VITASHELL2.02和工具箱1.64美化修复版（支持日历安装版）.vpk，复制到可移动磁盘。

9.打开vitashell气泡，安装enso_ex_v4.1_24x24_3.65_vs0.vpk或enso_ex_24x24-ur0.vpk和一键固化VITASHELL2.02和工具箱1.64美化修复版（支持日历安装版）.vpk。

10.主页打开气泡enso 3.65后，按o键，选择安装/重装enso-ex，按x键，按任意键重启，此时查看系统资讯，应该系统已是3.65变革o。

11.主页打开一键固化vitashell，选择第一项一键固化vitashell，2.02到系统，按o键执行，选择二项一键固化vita工具箱1.64到系统，按o键执行，执行完毕后选择第五项重构数据库，重启后可以删除原vitashell和h-encore了。将一键固化VITASHELL2.02和工具箱1.64美化修复第二版（自定义背景音乐）文件夹下中文主题配置文件文件夹下VitaShell压缩包解压，将解压的文件复制替换psv可移动磁盘VitaShell文件夹下所有文件。

以下为强迫症专用 如无强迫症可自行跳过

12.主页打开固化文件管理器气泡，usb连接电脑，按select按键连接电脑，将RW提权.vpk(bhqp)放入可移动磁盘，在psv固化文件管理器uxo文件见下找到进行安装，安装完毕后撕页退出，打开气泡rw提权，按o键运行后自动退出。（此时已经获得删除系统软件的权限）
13.主页打开固化文件管理器气泡，选择vs0/app/NPXSxxxxx(NPXSxxxxx皆为系统软件，想要卸载系统软件，直接删除某一个NPXSxxxxx文件夹即可，系统软件对照表如下，只能删除目录下的文件夹，其他不要删，删错成砖概不负责。)

一般删除的列表
Npxs10000
Npxs10001
Npxs10002
Npxs10003
Npxs10006
Npxs10009
Npxs10010
Npxs10012
Npxs10013
Npxs10014
Npxs10072
Npxs10094
按照自身想法清理完系统软件后 在主页可以删除提权气泡。如果显示不正常，用工具箱重构数据库。

跳过强迫症环节的直接从这继续往下看

14.打开vita工具箱气泡，选择常用插件大全，安装001、002、003插件（其他插件按照自身需要选装，001、002、003必装）

15.解压[psvTF卡格式化工具]，如下图对tf卡进行操作，对tf卡写入完毕后退出软件。

16.对TF卡进行格式化 选择exfat格式。

17.psv主页打开固化文件管理器，连接USB线，按select按键连接电脑，打开可移动磁盘，全选所有文件复制到刚才格好的tf卡。

18.将插好卡套的TF卡插进psv游戏卡插槽处。

19.打开vita工具箱气泡，选择tf卡套卡插件，选择第二项(tf卡>ux0，原装卡>uma0)。

20.打开固化文件管理器，按start键，将第二项选为外置tf卡套，撕页退出，打开设定，选择系统——自动启动设定，里面所有选项全部不勾选（关闭自动更新）至此破解降级固化完成。

HENkaku设置不见了的机友可以打开h-Encore程序，选择第二个选项重新安装变革破解，然后选择第一个选项退出。这时候打开设定就能看到了。
